# 🌟 Quote Collection

This is a shared repository where contributors submit one of their favorite quotes and explain why it’s meaningful to them.

## ✅ How to Contribute

1. Fork this repository.
2. Create a feature branch to add your quote (the date should be the start date of the cohort!):
```
git checkout -b <your-full-name>-<MMDDYY>-quote
```
3. Navigate to the quotes/ folder in the project.
4. Create a new markdown file named exactly like this:
`quotes/<your-full-name>-<MMDDYY>-quote.md`
5. Add your favorite quote using the example format below:
```
# My Favorite Quote

> "You miss 100% of the shots you don't take."  
> — Wayne Gretzky

**Why I Like It:**  
This quote reminds me that it's better to try and fail than to not try at all.
```
6. Commit and Push Your branch:
```
git add <your file>
git commit -m "Add quote file for <your full name> from the <MMDDYY> cohort"
git push origin <your branch>
```
7. Submit a Pull Request
